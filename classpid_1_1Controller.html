Detailed Description

template<class... C>
class pid::Controller< C >

A Controller class derived from any combination of P, I and D types.

Types that are inherited must also inherit from Base which guarantee the Control() function implementation for each component in the controller. Only one of each type can be used with the same typename parameter or a compilation error will occur.
